# Технический стек СтройХаб

## Frontend

| Технология | Зачем |
|-----------|-------|
| **React 19** | UI-фреймворк |
| **TypeScript** | Типизация |
| **Vite** | Быстрая сборка |
| **Tailwind CSS 4** | Стили |
| **React Router** | Навигация |
| **Zustand** | Состояние (готово к подключению) |
| **React Hook Form + Zod** | Формы и валидация |
| **date-fns** | Даты |
| **lucide-react** | Иконки |
| **react-hot-toast** | Уведомления |
| **Capacitor** | Android / iOS |

## Backend

| Технология | Зачем |
|-----------|-------|
| **Supabase** | PostgreSQL, Auth, Storage |
| **Row Level Security** | Безопасность данных |

## Интеграции

| Сервис | Зачем |
|--------|-------|
| **Telegram Web App** | Платформа и авторизация |
| **Yandex Vision API** | Распознавание чеков |
| **Yandex SpeechKit** | Голосовой ввод |
