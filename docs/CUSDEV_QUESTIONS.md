# Вопросы для CusDev (интервью с прорабами)

## Блок 1: Текущие инструменты

1. **Как вы сейчас ведёте учёт денег на объекте?**
   - Excel? Блокнот? Голова? Приложение?

2. **Как вы ставите задачи бригаде?**
   - Звонок, WhatsApp, Telegram, устно на объекте?

3. **Как заказчик узнаёт о прогрессе?**
   - Фото в мессенджере, звонки, никак?

4. **Сколько объектов ведёте одновременно?**
   - 1–2 / 3–5 / больше 5

## Блок 2: Боли

5. **Что бесит больше всего в текущем процессе?**
   - Потеря чеков, путаница в задачах, звонки заказчика, другое?

6. **Бывали перерасходы бюджета? Как узнавали?**
   - Сразу / в конце / не отслеживали

7. **Как фиксируете выполненные работы?**
   - Фото, акты, устно?

## Блок 3: Готовность платить

8. **Платили бы за приложение, которое экономит 2–3 часа в день?**
   - Да / нет / зависит от цены

9. **Какая цена кажется справедливой?**
   - 0 ₽ / до 1000 ₽ / 1500–3000 ₽ / больше

10. **Что должно быть в бесплатной версии, чтобы вы попробовали?**

## Блок 4: Функции СтройХаб

11. **Нужен ли заказчику доступ к прогрессу без звонков?**
12. **Нужно ли распознавание чеков камерой?**
13. **Нужен ли голосовой ввод для мастеров в перчатках?**
14. **Нужна ли заявка на материалы из приложения?**

## Как проводить интервью

- 15–20 минут, 3–5 прорабов
- Не продавать продукт — слушать
- Записывать дословные формулировки болей
- В конце показать прототип и спросить: «Чего не хватает?»
